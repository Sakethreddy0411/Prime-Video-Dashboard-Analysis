🎬 Prime Video Dashboard Analysis (Power BI)

📌 Project Overview

The **Prime Video Dashboard Analysis** project is a data visualization and analytics project built using Power BI.  
It focuses on analyzing Prime Video movies and TV shows data to uncover insights related to genres, ratings, countries, languages, and subscription types through an interactive dashboard.

---

🎯 Objectives

* Analyze Prime Video content distribution  
* Identify trends based on release years  
* Understand genre popularity  
* Study country-wise content availability  
* Analyze subscription types and language distribution  
* Build an interactive and user-friendly dashboard  

---

📊 Features

* 📈 Year-wise content trend analysis  
* 🌍 Country-wise distribution of shows  
* 🎭 Genre-wise content analysis  
* ⭐ Rating distribution visualization  
* 🗣 Language distribution (Donut Chart)  
* 💳 Subscription type breakdown  
* 🎛 Interactive slicers (Country, Year, Genre, Subscription Type)  
* 📊 KPI Cards for quick insights  

---

🛠 Tech Stack

* **Tool:** Power BI  
* **Data Processing:** Power Query  
* **Visualization:** Power BI Charts & Dashboard Design  

---

📂 Dataset

The dataset contains Prime Video content details such as:

* Title  
* Genre  
* Release Year  
* Country  
* Language  
* Rating  
* Subscription Type  
* Duration  

---

⚙️ Workflow

1. **Data Loading** – Imported dataset into Power BI  
2. **Data Cleaning** – Removed null values and formatted columns  
3. **Data Transformation** – Used Power Query for preprocessing  
4. **Data Modeling** – Created relationships and measures  
5. **Visualization** – Designed interactive dashboard with charts  
6. **Filtering** – Added slicers for dynamic analysis  

---

📈 Key Insights

* 📊 Content significantly increased after 2000  
* 🎭 Drama and Sci-Fi are among the most popular genres  
* 🌍 Certain countries contribute more content  
* 🗣 Multiple languages are supported, increasing reach  
* 💳 Subscription types are evenly distributed  

---

🚀 How to Use

1. Download the `.pbix` file  
2. Open in Power BI Desktop  
3. Use slicers to explore data interactively
